# Project Overview

InferKit is an independent pure-Dart package for OpenAI-compatible inference
servers. It is published to pub.dev for use by Dart and Flutter applications.

InferKit focuses on chat completions, model listing, streaming events,
reasoning extraction, and agent tool orchestration. It intentionally stays
independent from Flutter UI concerns and host application services.

For end-user examples, see [USAGE.md](USAGE.md). For the package boundary
split, see [BOUNDARIES.md](BOUNDARIES.md).
